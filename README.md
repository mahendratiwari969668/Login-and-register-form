   # 🔐 Animated Login & Register Form (Flip Effect)

A modern and responsive **Login & Register form** built using **HTML, CSS, and JavaScript** with a **3D flip animation**, **password visibility toggle**, and **neon hover effects**. Clean UI, lightweight code, and beginner-friendly project for web developers.

---

## ✨ Features
- 🔁 Smooth **3D Flip Animation** between login and register forms  
- 👁️ **Show / Hide Password** with toggle icons  
- 💡 **Neon Hover Effects** on inputs and buttons  
- 📱 **Fully Responsive Design** for all screen sizes  
- 🎨 **Modern Gradient Background** and elegant styling  
- 🧠 **Simple Vanilla JavaScript** (no frameworks)

---

## 🧩 Technologies Used
- **HTML5** – structure and layout  
- **CSS3** – styling and animations  
- **JavaScript (Vanilla)** – interactivity and flip logic  
- **Boxicons** – for icons  
- **Google Fonts (Poppins)** – for typography
